# Recipes

Short, problem-shaped guides for situations that come up when integrating Vatly into a non-trivial application. Each recipe is self-contained — read just the ones that apply.

- [Cannot customize owner model](cannot-customize-owner-model.md) — when the User / Organization / Tenant class is sealed (vendor package, third-party identity provider, immutable DTO) and you can't make it implement `BillableInterface` directly.
